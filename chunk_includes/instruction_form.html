Für diese Bewertungsaufgabe lesen Sie eine Reihe von deutschen Sätzen.

Für diesen Fragebogen bitten wir Sie zu beurteilen, ob die in blau geschriebene Sätze in Bezug auf den vorgehenden Kontext natürlich sein könnten.

Wenn Sie denken, dass ein Satz natürlich ist, geben Sie dem Satz eine hohe Punktzahl. Wenn Sie denken, dass ein Satz unnatürlich ist, geben Sie dem Satz eine niedrige Punktzahl.

Es gibt zwei Probeläufe, bevor das Experiment anfängt.
